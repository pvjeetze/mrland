# calcBMIshr

estimates population based on BMI share

## Usage

``` r
calcBMIshr(convert = TRUE)
```

## Arguments

- convert:

  Use raw data or interpolated data. Raw data should only be used for
  regressions.

## Value

List with a magpie object
