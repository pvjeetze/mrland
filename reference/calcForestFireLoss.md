# calcForestFireLoss

Calculate how much loss of forest area happens due to fire disturbances
based on FRA 2020 data

## Usage

``` r
calcForestFireLoss()
```

## Value

MAgPIE object with FRA 2020 forest fire area loss

## See also

[`readFRA2020`](https://rdrr.io/pkg/mrfaocore/man/readFRA2020.html)
