# calcForestFireShare

Calculates which share of forest land is lost due to forest fires

## Usage

``` r
calcForestFireShare()
```

## Value

MAgPIE object with FRA 2020 forest fire shares

## See also

[`readFRA2020`](https://rdrr.io/pkg/mrfaocore/man/readFRA2020.html)
