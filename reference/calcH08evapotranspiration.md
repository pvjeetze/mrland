# calcH08evapotranspiration

Calc evapotranspiration data for SSP cenarios in mm/month

## Usage

``` r
calcH08evapotranspiration(subtype = "H08:mri-esm2-0")
```

## Arguments

- subtype:

  Switch between different inputs

## Value

magpie object in cellular resolution
