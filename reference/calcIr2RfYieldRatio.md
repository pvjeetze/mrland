# calcIr2RfYieldRatio

Passes on the irrigated to rainfed yield ratio from AQUASTAT

## Usage

``` r
calcIr2RfYieldRatio()
```

## Value

MAgPIE object of yields

## See also

\[readAQUASTAT()\], \[convertAQUASTAT()\]
