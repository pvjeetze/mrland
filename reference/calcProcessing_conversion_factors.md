# calcProcessing_conversion_factors

Calculates global conversion factors from the FAOmassbalance_pre in
order to feed to MAgPIE for conversion of primary to secondary products

## Usage

``` r
calcProcessing_conversion_factors()
```

## Value

magpie object of conversion factors
