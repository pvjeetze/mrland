# calcTimberDemandExt

Calculates the demand of timber from FAO data (including intermediate
products).

## Usage

``` r
calcTimberDemandExt()
```

## Value

List of magpie objects with results on country level, weight on country
level, unit and description.

## See also

[`calcFAOmassbalance_pre`](https://rdrr.io/pkg/mrfaocore/man/calcFAOmassbalance_pre.html)
