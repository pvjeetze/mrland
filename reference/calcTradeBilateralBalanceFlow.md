# calcTradeBilateralBalanceFlow

Calculates balanceflows for trade and total mass balance to match
historical production

## Usage

``` r
calcTradeBilateralBalanceFlow(balanceflow = "trade")
```

## Arguments

- balanceflow:

  "trade" or "total"

## Value

Self import to dupply ratio
