# Convert PYieldCoeff data to ISO country level.

Convert PYieldCoeff data to ISO country level.

## Usage

``` r
convertPYieldCoeff(x)
```

## Arguments

- x:

  MAgPIE object containing data for fixed regional resolution

## Value

data as MAgPIE object disaggregated to country level
