# correctS4Nproject_input

corrects IMAGE inputs of total bioenergy (1st gen, 2nd gen and residues)
demand and co2 prices

## Usage

``` r
correctS4Nproject_input(x)
```

## Arguments

- x:

  magpie object

## Value

magpie object at country-level resolution

## See also

[`readSource`](https://rdrr.io/pkg/madrat/man/readSource.html)
