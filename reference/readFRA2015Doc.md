# Read FRA2015Doc

Read-in an FRA data from 2015 (forest resource assessment).

## Usage

``` r
readFRA2015Doc(subtype)
```

## Arguments

- subtype:

  data subtype.

## Value

magpie object of the FRA 2015 data

## See also

[`readSource`](https://rdrr.io/pkg/madrat/man/readSource.html)
