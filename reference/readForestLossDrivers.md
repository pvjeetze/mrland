# Read ForestLossDrivers

Read-in an Forest loss data (range 2001-2015 but only single annual
number her) (Source:DOI: 10.1126/science.aau3445 Table 1).

## Usage

``` r
readForestLossDrivers()
```

## Value

magpie object of the Curtis et al., 2018 Data

## See also

[`readSource`](https://rdrr.io/pkg/madrat/man/readSource.html)
