# readGLW3

Read the gridded livestock of the world 3 dataset.

## Usage

``` r
readGLW3(subtype = "Da")
```

## Arguments

- subtype:

  Subtype of file to be opened (either Da or Aw)

## Value

Magpie objects

``` r
if (FALSE) { # \dontrun{
```
