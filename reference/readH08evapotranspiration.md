# readH08evapotranspiration

Read evapotranspiration data

## Usage

``` r
readH08evapotranspiration(subtype = "H08:mri-esm2-0:historical")
```

## Arguments

- subtype:

  Switch between different inputs

## Value

MAgPIE objects with results on cellular level.
