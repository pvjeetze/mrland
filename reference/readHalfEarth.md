# readHalfEarth

Read in Half Earth data set containing conservation area for
biodiversity protection based on the Half-Earth approach

## Usage

``` r
readHalfEarth(subtype = "GLOBIO4")
```

## Arguments

- subtype:

  Data source to be read from

## Value

MAgPIE object containing biodiveristy protection area at cellular level
