# readOzoneYieldShock

read Ozone Yield Shock Data from the EAT-Lancet deepdive on Ozone shock
effects on crop yields.

## Usage

``` r
readOzoneYieldShock()
```

## Value

MAgPIE object with country level yield shock data for year 2050.

if (FALSE) { # \dontrun{
