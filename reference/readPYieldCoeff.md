# Read in coefficients for calculating pasture intensification

Read in csv file containing coefficients of linear regression for the
calculation of future pasture intensification dependent on animal
numbers

## Usage

``` r
readPYieldCoeff()
```

## Value

MAgPIE object

## See also

[`readSource`](https://rdrr.io/pkg/madrat/man/readSource.html)
