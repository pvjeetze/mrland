# Read Sathaye Forest

Read-in an Sathaye Forest data .csv file as magclass object

## Usage

``` r
readSathayeForest()
```

## Value

magpie object of the Sathaye Forest data

## See also

[`readSource`](https://rdrr.io/pkg/madrat/man/readSource.html)
