# readUrbanLandGao

Read gridded urban land, from Gao O'Neill and Jones SEDAC dataset,
https://sedac.ciesin.columbia.edu/data/set/ssp-1-8th-urban-land-extent-projection-base-year-ssp-2000-2100
\#nolint

## Usage

``` r
readUrbanLandGao()
```

## Value

magpie object of 2000-2100 urban land in Mha, in 10 year intervals
