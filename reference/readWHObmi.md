# readWHObmi

Reads in data on body mass index (BMI) recommendations from WHO
http://www.who.int/childgrowth/standards/bmi_for_age/en/
http://www.who.int/growthref/who2007_bmi_for_age/en/

## Usage

``` r
readWHObmi()
```

## Value

magpie object
